## How to Use

Log meaningful changes to Ahrefs, Screaming Frog, GSC, GA4, Shopify, Claude Code, MCP servers, or any tool an agent uses. Helps explain unexpected behavior and avoid duplicate investigation.

## Format

```
### YYYY-MM-DD - Tool and change

**Tool:** name.
**Change:** what changed, in one line.
**Source:** release notes URL or email.
**Impact on our workflow:** one or two sentences.
**Action:** what we did to adapt (or none needed).
```
