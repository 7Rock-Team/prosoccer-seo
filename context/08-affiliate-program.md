# 08 - Affiliate Program

_Read by: all agents. Operating doctrine, not analysis. Load-bearing for any backlink audit, disavow recommendation, or referring-domain quality assessment._

## Program Overview

7 Rock Marketing manages ProSoccer's affiliate program on **AWIN (Affiliate Window)**. ProSoccer owns the relationship; 7 Rock operates it. The active affiliate roster is maintained inside the AWIN publisher dashboard and is not mirrored in this repo.

## Why This Matters for SEO

Affiliate links show up in the backlink profile as third-party referring domains. An agent running a backlink audit without affiliate context will flag active AWIN affiliates as suspect and recommend disavowing them. Disavowing an active affiliate breaks a paid revenue channel.

## Verified and Unverified Referring Clusters

- **soccertop.com** (UAE domain). ~16M backlinks from a single UAE domain, representing over 90% of ProSoccer's total backlink profile. **Confirmed NOT an AWIN affiliate and NOT in PayAudit (Mike verified both sources).** Origin unknown. Most likely a scrape/mirror site or a negative SEO pattern. Disavow recommendation stands firm. Origin investigation is a separate follow-up.
- **Korean-anchor-text cluster.** Korean anchor text ("미국 축구용품 전문 종합쇼핑몰," "프로 사커," Korean renderings of adidas and puma) is specific enough to suggest intentional content, not scraped spam. Mike does not know of a formal Korean partner. **Has NOT yet been verified against AWIN or PayAudit; Mike will verify in a follow-up session.** Most likely an AWIN affiliate operating in Korean. No action until verified.

## Operating Rule

Any specialist agent producing a backlink audit, disavow file, or referring-domain quality assessment MUST cross-reference flagged domains against **both** the AWIN affiliate roster **and** PayAudit (7 Rock's pre-AWIN transaction-approval system) before surfacing the recommendation to Mike. Checking AWIN alone is insufficient. Pull both systems fresh at the start of each audit.

If a flagged domain appears in either system, it stays. If it appears in neither, assess on SEO merit. No exceptions.

## Reference

AWIN publisher dashboard is the source of truth. Not mirrored in this repo. Mike pulls the roster when an audit kicks off.
