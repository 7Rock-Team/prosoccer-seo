# Keyword Map

_Owned by Keyword Research agent once built. Until then, Master Strategist maintains._

## Purpose

Canonical map of target keywords to the URL that should rank for them. Prevents cannibalization. Prioritizes work.

## Format

| Keyword | Intent | Monthly Volume | Difficulty | Target URL | Status | Notes |
|---|---|---|---|---|---|---|
| _keyword_ | _info / comm / nav / trans_ | _number or tier_ | _easy / med / hard_ | _slug or full URL_ | _new / live / ranking / falling_ | _notes_ |

## Intent Key

- **Informational:** user wants to learn ("ball size u10")
- **Commercial:** user is comparing ("best keeper gloves 2026")
- **Transactional:** user is ready to buy ("buy Predator Elite 29")
- **Navigational:** user is looking for a brand/site ("prosoccer pasadena hours")
